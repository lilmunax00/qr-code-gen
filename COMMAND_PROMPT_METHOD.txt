                    🔧 SOLUTION: USE COMMAND PROMPT DIRECTLY
================================================================================

PROBLEM: Batch file closes before showing anything
SOLUTION: Run commands directly in Command Prompt (keeps window open!)

================================================================================
                        DO THIS RIGHT NOW (5 MINUTES)
================================================================================

STEP 1: OPEN COMMAND PROMPT
────────────────────────────────────────────────────────────────────────────
Press: Windows Key + R
Type: cmd
Press: Enter

Result: Black command prompt window opens


STEP 2: GO TO PROJECT FOLDER
────────────────────────────────────────────────────────────────────────────
Copy & paste this line:

cd "C:\Users\Lenovo\Documents\QR CODE GENERATOR SYSTEM"

Press: Enter

Result: You're now in the project folder


STEP 3: INSTALL HEROKU CLI
────────────────────────────────────────────────────────────────────────────
Copy & paste:

npm install -g heroku

Press: Enter

Result: Takes 2-3 minutes to install
You'll see: "added X packages"


STEP 4: VERIFY INSTALLATION
────────────────────────────────────────────────────────────────────────────
Copy & paste:

heroku --version

Press: Enter

Result: Should show: heroku/7.68.0 (or similar)

If you see version number → ✅ SUCCESS!
If "command not found" → Try steps 3-4 again


STEP 5: LOGIN TO HEROKU
────────────────────────────────────────────────────────────────────────────
Copy & paste:

heroku login

Press: Enter

Result: Browser opens
Login with your Heroku account
Return to Command Prompt


STEP 6: DEPLOY YOUR SYSTEM
────────────────────────────────────────────────────────────────────────────
Copy & paste:

DEPLOY_HEROKU.bat

Press: Enter

Result: Deployment starts
Follow the prompts
Choose unique app name
Wait 5-10 minutes


STEP 7: YOU'RE DONE! 🎉
────────────────────────────────────────────────────────────────────────────
You'll see: "DEPLOYMENT SUCCESSFUL"
Your app URL: https://your-app-name.herokuapp.com

Copy that URL!
Open in browser!
Your system is LIVE! 🚀

================================================================================
                        TROUBLESHOOTING
================================================================================

Issue: "npm: command not found"
Solution: Node.js not installed
  → Download: https://nodejs.org
  → Install it
  → Restart computer
  → Try npm install -g heroku again

Issue: "heroku: command not found"
Solution: npm install didn't work
  → Try again: npm install -g heroku
  → Or download from: https://devcenter.heroku.com/articles/heroku-cli

Issue: Deployment fails
Solution: Check error message carefully
  → Read the error
  → Google the error
  → Try again or contact Heroku support

Issue: Can't login to Heroku
Solution: Check your Heroku account
  → Create account at: https://www.heroku.com
  → Verify email
  → Try heroku login again
