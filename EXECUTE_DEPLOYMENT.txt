================================================================================
                        TROUBLESHOOTING
================================================================================

❌ "Heroku CLI not found"
   → Download: https://devcenter.heroku.com/articles/heroku-cli
   → Install it
   → Restart computer
   → Try deployment again

❌ "App name already taken"
   → Choose different name
   → Re-run deployment
   → Example: qr-paint-system-yourname

❌ Deployment failed with error
   → Check error message in terminal
   → Run: heroku logs --tail --app your-app-name
   → Look for MongoDB or build errors
   → Try deployment again

❌ App loads but buttons don't work
   → Refresh page (F5)
   → Wait 30 seconds
   → Try again

❌ QR doesn't work from phone
   → Phone needs internet
   → URL must show https://your-app-name.herokuapp.com
   → Try opening URL in phone browser first
   → Then try scanning
