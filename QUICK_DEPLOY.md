## 🆘 If Something Goes Wrong

### Check Logs
```bash

```

### Restart App
```bash
heroku restart --app your-app-name
```

### Verify MongoDB
Go to https://cloud.mongodb.com - check collections

### Verify Config
```bash
heroku config --app your-app-name
```
